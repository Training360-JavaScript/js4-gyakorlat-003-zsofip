# Feladatok

## NAGYON FONTOS!!!
- A könnyebb tesztelhetőség érdekében az elkészített függvényeket és változókat 
exportálni kell!
- Függvények esetén egy példa:
```javascript
export {
    objectsMerge,
}
```
- Változók esetén egy példa:
```javascript
export {
  firstName,
  lastName,
  job,
};
```
